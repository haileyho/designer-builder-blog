# Designer Builder Blog

A clean, minimal static blog website about design process, building, and the craft of making things that matter.

## Pages

- **index.html** — Homepage with hero section, featured post, post grid, and newsletter signup
- **post.html** — Individual blog post page with full article layout
- **style.css** — All styles (CSS variables, responsive layout, typography)

## Features

- Responsive design (mobile-friendly)
- Google Fonts: Playfair Display + Inter
- Sticky header with blur backdrop
- Hover animations on cards
- Newsletter signup form
- Clean article reading layout with blockquotes

## Deploy with GitHub Pages

1. Go to **Settings → Pages**
2. Set source to **Deploy from a branch**
3. Select **main** branch, **/ (root)** folder
4. Click **Save** — your site will be live at `https://haileyho.github.io/designer-builder-blog/`

## Customization

Edit `index.html` to change the blog posts, author name, and site title.  
Edit `style.css` CSS variables at the top of the file to change colors and fonts.
